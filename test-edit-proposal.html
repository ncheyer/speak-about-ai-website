Testing edit functionality for proposals

Steps to test:

  1. Navigate to http://localhost:3002/admin/proposals
  2. Click on the 3 dots menu for any draft proposal
  3. Click "Edit"
  4. Verify that all proposal data is loaded correctly
  5. Make changes and save

Expected behavior:

  • Proposal form should load with existing data
  • All fields should be populated (client info, speakers, services, etc.)
  • Saving should update the existing proposal, not create a new one
